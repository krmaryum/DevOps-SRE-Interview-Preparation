# Networking Interview MCQ Quiz

An interactive quiz designed to test and strengthen networking knowledge for DevOps and SRE interviews.

## Quiz Highlights

- 25 networking interview questions
- 25-minute timer
- 80% passing score
- Instant answer checking and short explanations
- Live progress and final score
- Questions and answer choices shuffled on every attempt
- Reattempt option for continued practice

## Topics Covered

TCP/IP, subnetting, routing, DNS, TCP and UDP, ports, firewalls, NAT, HTTP, TLS, proxies, load balancing, cloud connectivity, and network troubleshooting.
