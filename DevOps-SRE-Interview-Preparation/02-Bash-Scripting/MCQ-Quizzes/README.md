# Bash Scripting Interview MCQ Quiz

An interactive quiz designed to test and strengthen Bash scripting knowledge for DevOps and SRE interviews.

## Quiz Highlights

- 25 Bash interview questions
- 25-minute timer
- 80% passing score
- Instant answer checking and short explanations
- Live progress and final score
- Questions and answer choices shuffled on every attempt
- Reattempt option for continued practice

## Topics Covered

Bash execution, quoting, streams, positional arguments, conditions, loops, functions, arrays, strict mode, traps, security, debugging, and automation.
