# Linux Interview MCQ Quiz

An interactive quiz designed to test and strengthen Linux knowledge for DevOps and SRE interviews.

## Quiz Highlights

- 25 Linux interview questions
- 25-minute timer
- 80% passing score
- Instant answer checking and short explanations
- Live progress and final score
- Questions and answer choices shuffled on every attempt
- Reattempt option for continued practice

## Topics Covered

Linux architecture, permissions, processes, systemd, performance, storage, SSH, networking, scheduling, and troubleshooting.
