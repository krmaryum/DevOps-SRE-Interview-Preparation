# Kubernetes Interview MCQ Quiz

An interactive quiz designed to test and strengthen Kubernetes knowledge for DevOps and SRE interviews.

## Quiz Highlights

- 25 interview questions
- 25-minute timer
- 80% passing score
- Instant answer checking and short explanations
- Live progress and final score
- Questions and answer choices shuffled on every attempt
- Reattempt option for continued practice

## Intended Topics

Kubernetes architecture, workloads, scheduling, probes, networking, storage, RBAC, security, reliability, and troubleshooting.

## Purpose

Use this quiz to identify knowledge gaps, revise essential Kubernetes concepts, and prepare for DevOps Engineer, Cloud Engineer, Platform Engineer, Kubernetes Administrator, and SRE interviews.
