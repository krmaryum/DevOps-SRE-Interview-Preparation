# Docker Interview MCQ Quiz

An interactive quiz designed to test and strengthen Docker knowledge for DevOps and SRE interviews.

## Quiz Highlights

- 25 Docker interview questions
- 25-minute timer
- 80% passing score
- Instant answer checking and short explanations
- Live progress and final score
- Questions and answer choices shuffled on every attempt
- Reattempt option for continued practice

## Topics Covered

Docker architecture, images, Dockerfiles, containers, runtime operations, networking, storage, Docker Compose, security, reliability, and troubleshooting.
